# Proyecto Final Java
Proyecto Final – Estructuras de Datos en Java

Este proyecto es una aplicación educativa en Java 17 que implementa y demuestra múltiples estructuras de datos, organizadas en 3 cortes académicos.

Incluye:

✔ Código funcional
✔ Menús interactivos
✔ Ejemplos prácticos
✔ Visualización clara por consola
✔ Implementaciones completas
